# 🎨 موقع بصمة ديزاين

موقع احترافي لعرض خدمات التصميم والمواقع

## 📋 الأقسام الموجودة في الموقع:

1. **الرئيسية** - تعريف سريع بالخدمات والاتصال المباشر
2. **المصداقية** - عرض الصور والتقييمات من العملاء
3. **أعمالي** - عرض المشاريع السابقة مع الروابط
4. **الشهادات** - الشهادات والدورات والدراسات
5. **التواصل** - معلومات الاتصال والخاتمة

---

## 🚀 كيفية تحديث الموقع

### 1️⃣ إضافة صور إلى قسم المصداقية

في ملف `index.html`، ابحث عن الجزء:
```html
<div class="gallery">
    <div class="gallery-item placeholder">
```

استبدل أو أضف:
```html
<div class="gallery-item">
    <img src="path/to/your/image.jpg" alt="وصف الصورة">
</div>
```

### 2️⃣ إضافة مشاريع إلى قسم أعمالي

استبدل `placeholder` في قسم Portfolio:
```html
<div class="portfolio-item">
    <i class="fas fa-globe"></i>
    <h3>اسم المشروع</h3>
    <p>وصف المشروع المختصر</p>
    <a href="https://link-to-project.com" class="view-link" target="_blank">زيارة المشروع →</a>
</div>
```

### 3️⃣ إضافة شهادات

استبدل `placeholder` في قسم Certificates:
```html
<div class="cert-card">
    <i class="fas fa-certificate"></i>
    <h3>اسم الشهادة</h3>
    <p>اسم الدورة أو الجهة المانحة</p>
</div>
```

### 4️⃣ استخدام JavaScript للتحديث الديناميكي

إذا كنت تريد تحديث الصور والمشاريع ديناميكياً، افتح ملف `script.js` وغيّر الأسطر في النهاية:

```javascript
// إضافة مشروع
updatePortfolioItem(0, 'اسم المشروع', 'الوصف', 'https://link.com');

// إضافة صورة
updateGalleryImage(0, 'path/to/image.jpg');

// إضافة شهادة
addCertificate('اسم الشهادة', 'الوصف');
```

## 🎨 الألوان والتصميم

يمكنك تغيير الألوان في ملف `styles.css` بتعديل القيم في `:root`:

```css
:root {
    --primary-color: #FF6B9D;      /* اللون الرئيسي */
    --secondary-color: #C44569;    /* اللون الثانوي */
    --accent-color: #FFA06B;       /* لون التميز */
}
```
